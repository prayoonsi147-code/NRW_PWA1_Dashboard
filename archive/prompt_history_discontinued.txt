โครงสร้างหน้า manage.html:
  1. Header — ชื่อ + ปุ่มกลับ Dashboard + ปุ่ม Dark/Light toggle
  2. Upload Section (Tabbed) — Tab แยกหมวดข้อมูล + จำนวนเดือน
     - แต่ละ Tab: mode selector (อัตโนมัติ/ระบุเอง inline) + Drop Zone + คำแนะนำ (open) + ปุ่ม Upload
  3. จัดการข้อมูล — ปุ่มเปิดโฟลเดอร์ + Chips จัดกลุ่มตามปี (แยกสีตามหมวด)
  4. บันทึกการอัปโหลด — Log พร้อม timestamp

ฟีเจอร์สำคัญ:
  - Upload ซ้ำ = เขียนทับ + เตือนก่อน (⚠️ dialog สีแดง)
  - Error → ข้อความภาษาผู้ใช้ (ไม่แสดง technical error ตรง)
  - Dark mode sync ผ่าน localStorage('XXXX_theme')
  - Theme สีตรงกับ Dashboard หลัก
  - ปุ่มเปิดโฟลเดอร์ (localhost only)
  - รองรับเพิ่มหมวดข้อมูลอนาคต (เพิ่ม Tab)

Server (Flask) endpoints:
  - GET  /              → serve index.html
  - GET  /manage.html   → serve manage.html
  - GET  /api/ping      → health check
  - GET  /api/data      → ข้อมูลทั้งหมด (JSON)
  - POST /api/upload/XX → upload ไฟล์ (auto-detect + parse + save)
  - DELETE /api/data/XX/<mk> → ลบข้อมูลเดือน
  - POST /api/open-folder → เปิด File Explorer (localhost only)

1. สร้าง server.py — Flask server, port ไม่ซ้ำกัน
   - PR = 5000, Leak = 5001, GIS = 5002, Meter = 5003

2. สร้าง manage.html — ใช้ Design Pattern เดียวกัน เปลี่ยนแค่:
   - Theme สี (ตรงกับ Dashboard หลัก)
   - Tab หมวดข้อมูล (ตามประเภทข้อมูลของ Dashboard นั้น)
   - localStorage key สำหรับ Dark mode (ตรงกับ Dashboard หลัก)

3. manage.html ต้องมี redirect script ที่ <head>:
   <script>
     if (location.protocol === 'file:') {
       location.href = 'http://localhost:PORT/manage.html';
     }
   </script>
   ** ข้อสำคัญ: ถ้าเปิด manage.html เป็น file:// จะเรียก API ไม่ได้
   ต้อง redirect ไป localhost อัตโนมัติ **

4. Dashboard index.html — เพิ่มปุ่ม "จัดการแหล่งข้อมูล"
   - href="manage.html" (ลิงก์ปกติ)
   - อยู่ใน Header ข้าง Dark mode toggle

5. ปุ่ม "← หน้าหลัก" ใน Dashboard index.html:
   - ใช้ smart pattern: ถ้า file:// ใช้ href="../index.html"
     ถ้า localhost ใช้ fetch('/api/open-main') เปิดไฟล์ตรง
   - เพราะ Flask server serve เฉพาะ folder ตัวเอง
     ลิงก์ ../index.html ผ่าน Flask จะไม่ทำงาน

6. start_server.bat — อัปเดตให้มี option เปิด server ใหม่
   - port ไม่ซ้ำกัน
   - Option "Main + All" เปิดหน้าหลัก + server ทั้งหมด
