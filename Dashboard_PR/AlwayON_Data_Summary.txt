DATA FORMAT IN JS FILE
================================================================================

Structure: alwayONData[monthCode].branches[branchName]
Example:
  alwayONData["ม.ค.69"].branches["สาขาชลบุรี"] = {
    always_on: 0.8685404964337493,
    water_bill: 95725503.7,
    payment_total: 137259.0
  }

Month Codes:
- "ต.ค.68" = October 2568
- "พ.ย.68" = November 2568
- "ธ.ค.68" = December 2568
- "ม.ค.69" = January 2569

Helper Functions in JS file:
- getAlwaysOnValue(monthCode, branchName) - Get Always-On % for specific branch
- getJanuaryAlwaysOnRanking() - Get all January branches sorted by Always-On
